Multiple Approval Request Odoo Apps

All in one Approval Odoo Apps

Multi Level Approval Odoo App helps users to streamline the approval process in business. This app allows users to set up multiple request types, and multiple approvers for various request types this ensures that each approval is reviewed and approved by the appropriate person so the approval process is efficient and effective. This app is useful for businesses that require multiple levels of approval for different types of requests. The ability to customize the approval process based on request type and assign appropriate approvers can definitely help streamline and expedite the approval process.

Features

Approval Access Rights

User access rights are added for approval flow.

Setup Multiple Request Type

Manage Multiple approval request types.

Setup Multiple Request Approvers

Configure multiple minimum approvers for approval process.

Approval Mail Notification

Email notification on approve or refuse of approval request.
Demo Request Video Tutorial

Approval User Access Right

Users having "Approval: User" access rights can create and approve approval requests.

Approval Settings Access Right

Users having "Approval: Settings" access rights can create, edit and approve approval requests.

Approval Types

User can create multiple approval request types.

User can see configure fields and also add approvers for approval request.

User can enable "Mail Notification" option to get notification for approve, or refuse of approval request.

Create Request

User can create approval request by clicking "CREATE NEW REQUEST" button on specific approval request type.

User need to click on "SUBMIT" button to submit approval request.

User can see approval request is submitted.

Email Notification

User will get email notification on submit of approval request.

Approval Request To Review

Under "Approvals > To Review" menu user can see request to review.

Approve Request

Approval users can approve request by clicking "APPROVE" button.

As per added approvers on request type user can approve requests.

Approved Request

User can see approved request.

Email Notification

User will get email notification on request approval.
